# 历史文档归档说明

> 归档时间：2026-05-15

本目录保存项目早期设计、阶段计划、工程评审、开发过程记录和旧数据库说明。归档不是删除，它的目的只是避免历史材料继续混入上线正式文档。

## 目录说明

| 目录 | 内容 | 使用场景 |
|---|---|---|
| `superpowers/` | 早期产品设计、阶段规格和实现计划 | 追溯产品和技术决策来源 |
| `reviews/` | 2026-05-07 工程 Review | 追溯已完成或已废弃的问题 |
| `开发过程记录/` | Pro、Max、AI 情报台等阶段实施和验收过程 | 排查历史问题、查找改造背景 |
| `数据库历史材料/` | 旧数据库说明和过渡迁移脚本 | 查找旧库升级背景 |

## 注意

- 当前正式产品用户端是 `info-mvp`，历史文档中大量 `info-max` 描述只代表当时状态。
- 历史文档中的旧路径，如 `docs/开发文档`、`docs/数据库`、`D:\ai-coding`，不再作为当前路径使用。
- 当前数据库完整结构和初始化数据以 `info_aggregation/sql/mysql8_init.sql` 为准。
- 当前上线部署以 [../ops/部署上线手册.md](../ops/部署上线手册.md) 为准。
