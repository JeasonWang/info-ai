# Pro 收尾闭环实施计划

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 打通 Pro 第一验收版的管理闭环，让管理员可以在 `info-admin` 中触发采集、事件重建、质量刷新和低质量治理，并由 `info-serve` 统一鉴权、审计和转发。

**Architecture:** `info-admin` 只负责页面交互；`info-serve` 作为管理 API 入口，新增管理动作服务并代理到 `info_aggregation`；`info_aggregation` 继续负责实际采集、质量治理和事件重建。所有管理动作继续经过管理员鉴权和审计中间件。

**Tech Stack:** Go `net/http`、Vue3 + TypeScript + Vite、MySQL、FastAPI 采集服务。
