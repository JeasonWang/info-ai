# 首页移动端体验优化 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 将首页移动端改为“信息流优先 + 粘性频道”的轻量体验，让用户更快看到热点事件，并能在长列表中快速回到顶部。

**Architecture:** 保持现有 Vue 组件结构，不引入新状态管理。`HomeView.vue` 负责顶部状态、搜索、频道导航和回到顶部状态；`EventList.vue` 负责精简事件卡片展示；`styles.css` 负责移动端优先视觉布局和响应式适配。

**Tech Stack:** Vue 3 `<script setup>`、Vue Router、Vitest、Vue Test Utils、CSS 响应式布局。

- [ ] **Step 3: Implement HomeView scroll state**

Update the script imports and logic:

```ts
import { computed, onBeforeUnmount, onMounted, ref } from 'vue'

const showBackToTop = ref(false)

function handleScroll() {
  showBackToTop.value = window.scrollY > 360
}

function scrollToTop() {
  window.scrollTo({ top: 0, behavior: 'smooth' })
}

onMounted(() => {
  loadHome()
  handleScroll()
  window.addEventListener('scroll', handleScroll, { passive: true })
})

onBeforeUnmount(() => {
  window.removeEventListener('scroll', handleScroll)
})
```

Replace the existing `onMounted(loadHome)` call with the combined `onMounted` block above.
